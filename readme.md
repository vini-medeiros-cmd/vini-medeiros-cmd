# Vinícius Medeiros

**Engenheiro de Software** · Automação de Processos · Integração de Sistemas · Back-end

Construo sistemas que eliminam trabalho manual — desde automações internas até integrações entre plataformas corporativas. Combino capacidade técnica (Python, APIs REST/SOAP, Flask) com visão de negócio e gestão de KPIs, transformando processos manuais em soluções escaláveis, documentadas e orientadas a dados.

Atuo há alguns anos numa empresa de gestão em saúde ocupacional e segurança do trabalho, onde lidero a estruturação de processos e automações internas — hoje buscando migrar minha atuação cada vez mais para o desenvolvimento full stack.

## O que eu construo

| Área | Exemplos |
|---|---|
| **Automação** | Scripts em Python que eliminam tarefas manuais e repetitivas, com credenciais e configuração isoladas em `.env` |
| **Integração de sistemas** | Conectores entre APIs corporativas — REST e SOAP/WebServices, ERPs e CRM |
| **Back-end** | Plataformas web internas com Python/Flask, PHP e Node.js; APIs e banco (Postgres) com Supabase |
| **Front-end / Full stack** | Interfaces com HTML/CSS/JavaScript puro, React e Next.js, incluindo apps mobile empacotados com Capacitor |
| **BI & Dados** | Dashboards, KPIs e indicadores operacionais no Power BI |

## Projetos em destaque

| Projeto | Stack | Descrição |
|---|---|---|
| **Hub de automações interno** | Python · Flask · JavaScript · HTML/CSS | Plataforma web centralizada que orquestra automações internas (relatórios, processamento de XML, divisão de planilhas) com interface drag-and-drop, execução em tempo real via SSE e download de resultados — substituiu a execução manual de scripts avulsos, reduzindo o tempo operacional em torno de 70%. |
| **Automação de relatórios de saúde ocupacional** | Python · REST · SOAP · openpyxl | Script que gera relatórios mensais em Excel para dezenas de empresas-cliente, integrando duas APIs corporativas distintas (leitura via REST, gravação via SOAP) com mapeamento personalizado de campos clínicos. |
| **Validação diária de exames por agenda** | Python · REST API · HTML | Rotina de validação automática que cruza agenda, pedidos e resultados de exames, gerando relatório HTML colorido entregue diariamente às equipes operacionais — eliminou checagens manuais recorrentes. |
| **Plataforma de estatísticas de futebol** | Next.js · TypeScript · Supabase (Postgres/Auth) · Tailwind · Vitest | MVP com médias históricas de jogadores do Brasileirão (chutes, faltas, cartões, gols/assistências), com filtro por time e mando de campo. Ingestão diária de dados via job agendado (Vercel Cron), testes automatizados e migrations de banco versionadas. |
| **Calculadora de IMC** | HTML · CSS · JavaScript | Calculadora com validação em tempo real, classificação por faixa (OMS) e histórico salvo em `localStorage`. |
| **App de fitness pessoal** | React · Vite · Capacitor | Aplicação PWA empacotada também como APK Android, para acompanhamento de treinos e metas pessoais. |

> Os projetos corporativos acima envolvem sistemas internos de terceiros — o código roda em repositórios privados. Os detalhes técnicos e a lógica de automação estão descritos aqui.
